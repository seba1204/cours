\section{Complexité}
\begin{enumerate}[(a)]
  \item \q{Montrer que la complexité du tri par insertion est de $n^2$
          comparaisons dans le pire des cas.}

        Il y a deux boucles imbriquées l'une dans l'autre. Dans le pire des cas, elles
        font chacunes $n$ tours.
        Donc la complexité est $n^2$.
\end{enumerate}
