\section{Exemple 1}

On donne le code suivant.
\begin{code}
  A = [k**2 for k in range(5)]
  B = [k**2 for k in range(10, 15)]
  C = A.extend(B) # Concatene B a A
  del A, B
\end{code}
\begin{enumerate}
  \item \q{Que contient la liste} \inlineCode{C}\q{et pourquoi ?}\\
        La liste \inlineCode{C} est vide, ce n'est même pas une liste, car la méthode \inlineCode{extend} pour une \inlineCode{list} ne retourne rien.
  \item \q{Comment modifier ce code pour qu'il remplisse son rôle ?} \\
        Il faut modifier la ligne $4.$ par \inlineCode{C = A + B}.
\end{enumerate}
